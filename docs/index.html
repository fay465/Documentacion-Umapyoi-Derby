Examples

Asciimath block

A matrix followed by some square roots can be written as

\$[[a,b],[c,d]]((n),(k)) sqrt(4) = 2 sqrt(9) = 3\$

Inline in a paragraph

Some text might suddenly need to show some matrix calculations, such as \$[[a,b],[c,d]]((n),(k))\$.

Inline, using attribute substitution

If you are not interested in exciting formulae such as \$[[{a},{b}],[c,d]]((n),(k))\$, perhaps you would find the more mundane \${sqroot}(4) = 2\$ or even \${expr}\$ more to your taste.

Lists

  • A matrix can be written as \$[[a,b],[c,d]]((n),(k))\$.

    • A square root looks like \$sqrt(4) = 2\$

  • Another square root looks like \$sqrt(9) = 3\$
